# PyModbus

Script to test the ModbusTCP connection.
Install the requirements:

```bash
pip install -r requirements.txt
```

Create your `.env` file with your settings:

```txt
IP=xxx.xxx.xxx.xxx
PORT=502
ID=1
```

Then test the connection and read some values:

```bash
python main.py
```
